Alpha v9.1.0 - 2021-08-28

  • Changes
    • Updated localization
    • Choosing a button in the menu now has a new animation
    • Going back in the menu via the keyboard shortcut will display the animation as if you clicked the back button
    • The Tetra-X Initial Rotation sound has been replaced with a less annoying variant
    • The hold animation has been modified
    • Block out can now be dodged with initial rotation and initial hold
      • Note that you can also IRS/IHS into a block out, though doing so with IRS is harder since it applies wallkicks in Tetra Legends
    • Changed the modifier keys for Handheld and Combo Challenge to be only SHIFT to make it easier to active them on retry
    • Handheld now has a starting level option
  • Fixes
    • Freezing in Night of Nights can no longer be cancelled with a hold
    • The pregame music is no longer cut off by going to an option selection menu
    • Tucking pieces in Retro Console mode should work again, it may still not be 100% accurate however
    • The controls menu should no longer create random errors when navigating with the mouse
    • Made it harder to find Night of Nights mode by accident

Alpha v9.0.0 - 2021-01-27

  • This may be the final update I push to Tetra Legends, aside from any other bugfixes I may want to include. Thank you all for playing! I hope to surprise you with new projects in the future.
  • New Rotation Systems
    • ARS-X
      • Custom mirrored ARS-Ti with other slight modifications
      • It's generally more permissive than ARS-Ti, so it may not be desired by core arcade players
      • Best suited for guideline players who want a more arcade experience
  • Changes
    • Survival
      • The mode uses a PPT-oriented garbage system instead of 'change every x lines'.
    • Tetra-X Pro
      • You can now select a starting level
      • The music switches to Master (Almost Clear!) once the hold queue disappears
    • Menu
      • Every mode, even if there are no settings, now has its own submenu
      • The submenus for modes:
        • now show the name of the mode at the top
        • now play a pre-game drum & bass mix of their music tracks while on the submenu
          • will use a treble-only reverb mix if the track wasn't arranged for the game
        • now default to the 'start' button so you can spam 'accept' to skip past the game's settings
    • Updated localizations
  • Bugfixes
    • Ready, Go! should no longer have interference with sound & voice banks.
      • Modders, this is a property in the sound's info.json file. Check T99 or TGM for an example.
